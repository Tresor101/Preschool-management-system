# Secondary School Management System

Welcome to the **Fiontonton Secondary School** Management System!

## Opening in Visual Studio Code

This project is configured to work seamlessly with Visual Studio Code. Follow these steps:

### 1. Prerequisites

- Install [Visual Studio Code](https://code.visualstudio.com/)
- Install [Node.js](https://nodejs.org/) (optional, for package management)

### 2. Open the Project

1. Open VS Code
2. Click on **File** â†’ **Open Folder**
3. Navigate to this project directory and select it
4. VS Code will automatically detect the `.vscode` configuration

### 3. Install Recommended Extensions

When you open the project, VS Code will prompt you to install recommended extensions. Click **Install All** to get:

- **Prettier** - Code formatter
- **Live Server** - Launch a local development server with live reload
- **Auto Rename Tag** - Automatically rename paired HTML tags
- **Color Highlight** - Highlight colors in your code
- **HTML CSS Class Completion** - CSS class name completion for HTML
- **HTML CSS Support** - CSS support for HTML documents
- **CSS Peek** - Peek into CSS definitions
- **Tailwind CSS IntelliSense** - Tailwind CSS autocomplete

### 4. Run the Project

#### Option 1: Using Live Server (Recommended)

1. Right-click on `index.html` in the Explorer
2. Select **"Open with Live Server"**
3. Your default browser will open with the project running at `http://localhost:5500` (default Live Server port)
4. Changes will automatically reload in the browser

#### Option 2: With Debugging (Chrome/Edge)

1. First, start Live Server (see Option 1)
2. Press `F5` or click **Run** â†’ **Start Debugging**
3. Select **"Open with Chrome (Live Server)"** or **"Open with Edge (Live Server)"**
4. The page will open with debugging capabilities enabled

#### Option 3: Direct File Opening

1. Press `F5` or click **Run** â†’ **Start Debugging**
2. Select **"Open index.html (Direct)"** from the debug configurations
3. The page will open directly in your browser without a local server

### 5. Features

The VS Code configuration includes:

- **Auto-formatting** on save using Prettier
- **Auto-save** after 1 second delay
- **HTML/CSS validation** enabled
- **Launch configurations** for Chrome and Edge browsers
- **Workspace settings** optimized for web development
